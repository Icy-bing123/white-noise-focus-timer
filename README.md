# 白噪音专注倒计时应用

## 🎯 项目简介
一个结合白噪音播放与可视化倒计时的专注力提升工具，帮助用户在学习、工作时集中注意力。

## ✨ 功能特点
- 🎵 四种白噪音选择：雨声、海浪、篝火、咖啡馆
- ⏱️ 可自定义专注时长
- 📊 可视化进度条，颜色随剩余时间变化
- 🔊 实时音量控制
- 📱 响应式设计，支持手机和电脑

## 📁 文件说明
- `main.html` - 主页面文件
- `audio-player.js` - 音频播放器逻辑
- `sounds/` - 音频文件
- `images/` - 背景图片

## 🛠️ 技术栈
- HTML5 + CSS3 + JavaScript (ES6)
- Font Awesome图标库
- Audio API音频控制
- CSS变量动态背景
